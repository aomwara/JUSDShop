Versions:
Node.js version: v12.18.2,
Solidity version: >=0.4.22 <0.8.0, pragma experimental ABIEncoderV2,
web3 version: : 1.3.1,
Truffle version: : v5.1.59,
Ganache version: 2.5.4

Please open a terminal at the folder of this project, make sure you have installed npm, nodejs, truffle, Ganache and Metamask.

1. setup:

npm install
truffle compile
truffle migrate

2. run:
npm run start
or
yarn start


Once the program starts, you can start browsing items and make purchase, or you can review your previous orders.
